Discuss the scope of var, let, and const

scope of var : it has function-level scope.we can access them within the function in which they are declared .variables are subject to hoisting, which means they are moved to the top of their containing function or global scope during compilation. This can sometimes lead to unexpected behavior.

let and const Scope : We saw var has function level scope but Variables declared with let and const have block-level scope. This means they are only accessible within the block in which they are declared.

Tell us the use cases of null and undefined

undefined : If a variable is declared but the value is not assigned, then the value of that variable will be undefined. For example,
let name;
console.log(name); //output is : undefined

null : In JavaScript, null is a special value that represents an empty or unknown value. For example,
let number = null
The code above suggests that the number variable is empty at the moment and may have a value later.

What do you mean by REST API?

API stands for Application Programming Interface. An API establishes a connection between programs so they can transfer data.
REST stands for REpresentational State Transfer . A RESTful API is an architectural style for an application program interface (API) that uses HTTP requests to access and use data. That data can be used to GET, PUT, POST and DELETE data types, which refers to the reading, updating, creating and deleting of operations concerning resources.
We are able to communicate with servers using the HTTP protocol. With these protocols, we can Create, Read, Update and Delete data â€“ otherwise known as CRUD operations.

The most commonly used methods are:

  • GET: The get method is used to Read data on the server.
  • POST: The post method is used to Create data.
  • PATCH/PUT: The patch method is used to Update data.
  • DELETE: The delete method is used to Delete data.
